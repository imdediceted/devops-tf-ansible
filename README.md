# Terraform + Ansible (Yandex Cloud)
Папка `terraform` — манифесты (VPC, 3 ВМ).  
Папка `ansible` — playbook и роли (common, web, proxy).  
Прокси слушает :3000 и отдаёт страницу «Привет, это задача Александра».
